# Personal Website — Jianting Shi (史健廷)

深色风格个人主页，基于简历内容生成。

## 样式特点
- Deep Dark Mode 配色
- Apple 风格简洁大气设计
- 液态玻璃效果（glassmorphism）
- 视差滚动粒子背景
- 霓虹紫呼吸灯动效
- 便当盒卡片式项目布局
- 滚动淡入动画

## 结构
- `index.html` — 单页完整网站
- `assets/` — 资源文件夹（暂无外部资源）

## 内容章节
1. Hero — 首屏个人介绍
2. About — 关于我
3. Education — 教育背景（时间线）
4. Experience — 工作经验
5. Projects — 项目经历（Bento Grid）
6. Skills — 技术能力
7. Publications — 代表性论文
8. Awards — 荣誉奖项
9. Summary — 个人总结
